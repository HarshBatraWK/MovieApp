About This Project

This project is a Netflix-inspired movie streaming platform built using modern web technologies. It is intended as a full-stack practice application that demonstrates skills in frontend and backend development, RESTful API design, responsive UI/UX, and integration with real-world development tools and frameworks.

Tech Stack

  • Frontend: Angular + Tailwind CSS
  • Backend: C# ASP.NET Core Web API
  • ORM: Entity Framework Core
  • Database: Microsoft SQL Server

Features

  • User authentication (login/signup)
  • Movie browsing and search
  • Movie watching with Dynamic Content loading
  • Watchlist and favorites
  • Responsive UI with mobile support
  • Admin panel for content management
  • REST API integration
